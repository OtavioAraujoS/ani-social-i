# 🌌 Ani Social (Otaku Sync)

**Ani Social** é uma plataforma social minimalista, moderna e de alta fidelidade projetada para a comunidade de anime. Combinando uma estética "Cyber-Lofi" com princípios de design "Nocturnal Precision", ela serve como um hub digital onde os residentes podem acompanhar seu histórico de episódios, participar de discussões profundas e manter-se conectados.

## 🎨 Filosofia de Design: Santuário Cyber-Lofi

A identidade visual do projeto é construída sobre três pilares fundamentais:

- **Nocturnal Precision**: Uma interface focada em modo escuro profundo, utilizando camadas tonais e destaques sutis.
- **Kintsugi Azure**: Celebrando a "beleza na imperfeição" através de detalhes inspirados em ciano e dourado e elementos de UI glassmórficos.
- **Regra Sem Linhas (No-Line Rule)**: Usando elevação e diferenças tonais em vez de bordas rígidas para criar uma experiência fluida e imersiva.

## 🚀 Principais Funcionalidades

- **Dashboard Centralizado**: Uma visão unificada da sua biblioteca de animes, atividades da comunidade e usuários.
- **Feed de Discurso**: Um espaço para teorias aprofundadas, notícias e enquetes da comunidade com layouts de cartões de alta fidelidade.
- **Status dos Residentes**: Acompanhamento em tempo real do que outros membros do santuário estão assistindo no momento.
- **Catálogo de Animes**: Um sistema simplificado de cadastro para adicionar novos títulos à biblioteca global da comunidade.

## 🛠️ Stack Tecnológica

- **Framework**: [Next.js 16](https://nextjs.org/)
- **Biblioteca**: [React 19](https://react.dev/)
- **Estilização**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Tipagem**: [TypeScript](https://www.typescriptlang.org/)
- **Ícones**: [Lucide React](https://lucide.dev/)
- **Gerenciamento de Estado**: [Zustand](https://github.com/pmndrs/zustand)
- **Formulários**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

## 📦 Primeiros Passos

### Pré-requisitos

- Node.js (v20+)
- pnpm (recomendado) ou npm/yarn/bun

### Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/OtavioAraujoS/ani-social-i.git
   ```

2. Instale as dependências:

   ```bash
   pnpm install
   ```

3. Execute o servidor de desenvolvimento:

   ```bash
   pnpm dev
   ```

4. Abra [http://localhost:3000](http://localhost:3000) para acessar o dashboard.

## 🐳 Suporte ao Docker

Você também pode executar o projeto usando Docker:

```bash
pnpm docker:dev
```
